Block-Level Elements

Block-Level Elements have below characteristic:

  • The element always start from the new row
  • We can modify it's attribute by using CSS
  • They are often used as container, cover others elements.
  • For example: p, H1-H6, ul, ol, etc.


Inline Elements

Inline Elements have below characteristic:

  • This element are within the row, unless it reach the end of the row
  • The width & height are depending on the content itself. Mean their attribute can't be changed
  • For example: span, a, em, b, i, img

Inline-Block Elements

Inline-Block Elements is a special element provided by CSS.

  • Use CSS, input display: inline-block to enable attribute edit for inline elecments
  • Applicable for inline elements, others default such as input, button
  • input, button by default can edit attribute
